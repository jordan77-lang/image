Accessibility Best Practices

Alt Text Guidelines

  • Keep it concise (typically under 150 characters)
  • Describe the content and function of the image
  • Don't start with "image of" or "picture of"
  • Include text that appears in the image
  • Consider context and purpose

Long Descriptions

  • Use for complex images like charts, diagrams, and infographics
  • Provide detailed information that conveys the same meaning
  • Structure the description logically
  • Include data points and relationships
  • Make it accessible to screen reader users

When to Use Each

  • Alt text: All images need alt attributes
  • Long description: Complex images that need more detail
  • Decorative images: Use empty alt attribute (alt="")
  • Text in images: Include all text in alt or long description
